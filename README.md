# Reto Literalura - Oracle One Next Education 2026

En litertura encontrarás diversas funciones que te permitirán crear tu catálogo de tus películas favoritas.
Solo necesitas buscar tus libros de interés y la magia aparecerá :)

---

## Como usar Literalura
El programa se ejecuta directamente en la consola donde podrás encontrar las siguientes opciones: 

                    |1 - Buscar libro por título                    |
                    |2 - Listar libros registrados                  |
                    |3 - Listar autores registrados                 |
                    |4 - Listar autores vivos en un determinado año |
                    |5 - Listar libros por idioma                   |
                    |                                               |
                    |0 - Salir                                      |
Es importante tener en cuenta que **solamente las opciones del 2 al 5 pueden ser ejecutadas con los libros buscados anteriormente** (opción 1).
